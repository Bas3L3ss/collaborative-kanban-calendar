# Collaborative Kanban Calendar - Frontend

A modern frontend application for a collaborative Kanban and calendar-based project management platform inspired by Trello, Jira, and Linear.

The goal of this project is to build a clean, scalable, and responsive user interface that enables teams to organize tasks, visualize project progress, and manage deadlines through both Kanban boards and calendar views.

## Features

- Authentication UI
- Workspace Management
- Kanban Boards
- Task Management
- Drag-and-Drop Interface
- Calendar View
- Notifications
- Activity History
- Responsive Design
- Realtime Collaboration Ready

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- React Router
- TanStack Query
- Zustand
- dnd-kit
- FullCalendar

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
